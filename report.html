Звіт про створення сайту «Computer Store»

1. Мета та завдання проєкту

Основна мета створення вебсайту «Computer Store» — розробити повноцінний, сучасний і зручний інтернет-магазин комп’ютерної техніки, що дозволяє користувачам переглядати каталог товарів, отримувати детальну інформацію про продукцію, а також реєструватися для взаємодії з платформою.

Сайт має виконувати роль демонстраційного прикладу структури сучасного e-commerce проєкту, який побудований на основі HTML, CSS, Flexbox і адаптивного дизайну. Його створення дає змогу закріпити знання з веброзробки, організації структури проекту, роботи з репозиторіями GitHub і публікації сайтів у відкритий доступ.

Завдання, які було поставлено в рамках проєкту:

  • Розробити структуру сайту з кількома сторінками (головна, каталог, про нас, контакти, реєстрація).
  • Реалізувати сучасний дизайн з неоновими відтінками та темною кольоровою схемою.
  • Створити каталог товарів із різними категоріями та можливістю додавання до кошика.
  • Забезпечити адаптивність інтерфейсу для мобільних пристроїв за допомогою медіа-запитів.
  • Здійснити публікацію сайту на GitHub Pages.

2. Використані технології

Під час створення сайту було використано сучасні технології веброзробки:

  • HTML5 — основна мова розмітки сторінок, що визначає структуру контенту.
  • CSS3 — використано для стилізації елементів, створення анімацій та візуальних ефектів.
  • Flexbox — система розмітки, що забезпечує правильне вирівнювання елементів незалежно від розміру екрана.
  • Grid Layout — використано в каталозі для формування гнучкої сітки товарів.
  • Media Queries — для адаптації сайту під різні пристрої (телефони, планшети, ПК).
  • Git — контроль версій, збереження історії змін.
  • GitHub Pages — публікація сайту у відкритий доступ.
  • Visual Studio Code — середовище розробки, у якому здійснювалась верстка та робота з терміналом.

3. Структура сайту

Вебсайт має чітко визначену структуру, що забезпечує просту навігацію та логічну організацію контенту:

  • index.html — головна сторінка з банером, промо-блоками та розділом “Чому ми?”.
  • catalog.html — сторінка каталогу, що містить товари, поділені за категоріями (комплектуючі, ноутбуки, монітори, аксесуари).
  • about.html — розділ з інформацією про компанію, її історію, місію та цілі.
  • contacts.html — контактна інформація, електронна пошта, соціальні мережі, форма для зворотного зв’язку.
  • register.html — сторінка реєстрації користувачів із простою формою введення даних.

Усі сторінки об’єднані через навігаційне меню у верхній частині сайту. Навігація реалізована гнучко, тому виглядає однаково добре на будь-яких екранах.

4. Дизайн та стилізація

Дизайн сайту побудований у стилі «неоновий кібер-мінималізм». Основні кольори — темно-сірий (#0b0c10) та яскраво-бірюзовий (#00fff5). Така кольорова схема створює сучасний вигляд, нагадує геймерську тематику та підкреслює технічний напрямок сайту.

Шрифти підібрані максимально прості для читання — Segoe UI. Використані ефекти наведення (hover), плавні тіні та переходи для створення “живого” інтерфейсу.

Елементи розташовані за допомогою Flexbox для гнучкого позиціювання та Grid для каталогу, що автоматично підлаштовується під кількість елементів.

5. Реалізація адаптивності

Адаптивність сайту реалізовано за допомогою медіа-запитів (@media), які змінюють розташування, розміри шрифтів і структуру сторінки під роздільну здатність екрана.

Основні принципи адаптивності:

  • Меню переходить у вертикальний вигляд при ширині екрану менше 768px.
  • Каталог перебудовується з 4 колонок до 2 або 1.
  • Зменшуються зображення, відступи та розмір кнопок.
  • Відключається надмірна тінь або анімація для мобільних пристроїв.

Таким чином сайт залишається зручним та читабельним на всіх пристроях, від великих моніторів до смартфонів.

6. Додавання товарів та взаємодія

У каталозі реалізовано систему карток товарів. Кожна картка містить назву, короткий опис, ціну та кнопку “Додати в кошик”. Ця структура створена універсально, що дозволяє легко додавати нові товари без зміни структури сайту.

Код картки товару базується на гнучкому шаблоні HTML, де всі елементи мають власні класи, що спрощує їх стилізацію. Надалі можна буде підключити JavaScript для реалізації динамічної роботи кошика.

7. Використання Git та публікація на GitHub Pages

Проєкт створювався у середовищі Visual Studio Code. Для збереження історії змін і публікації використовувалась система контролю версій Git.

Основні команди, які застосовувались у терміналі:

git init
git add .
git commit -m "Initial version of website"
git branch -M main
git remote add origin https://github.com/Flom1xx999/computer-store.git
git push -u origin main

Після цього сайт було опубліковано за допомогою GitHub Pages. Це дозволяє переглядати сайт онлайн без необхідності запуску локального сервера. Також це спрощує демонстрацію роботи проєкту викладачу.

8. Висновки

У ході роботи над проєктом «Computer Store» було реалізовано повноцінний вебсайт із сучасним дизайном, логічною структурою, адаптивністю та можливістю подальшого розвитку. Створення сайту дало змогу опанувати важливі навички: верстку, стилізацію, організацію коду, публікацію на GitHub.

Отримано практичні знання щодо:

  • побудови структури сайту;
  • роботи з HTML, CSS, Flexbox, Grid та Media-запитами;
  • створення привабливого адаптивного дизайну;
  • публікації проектів у відкритий доступ;
  • використання системи контролю версій Git.

У подальшому планується додати функціонал на JavaScript: реєстрацію користувачів, інтерактивний кошик, фільтри товарів і систему збереження даних. Таким чином проєкт може перерости у справжній повнофункціональний інтернет-магазин.
